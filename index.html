AI Recipe Generator

    Serverless Web Application using Generative AI AWS Amplify and Amazon Bedrock Application Understanding the Core Components:
      Frontend:
      • HTML-based web interface for ingredient input
      • Amplify Hosting for frontend deployment
      Backend:
      • Amplify API (GraphQL) for handling frontend requests
      • AWS Lambda functions triggered by API requests
      • Amazon Bedrock for accessing large language models (e.g., Claude 3 Sonnet)
    Data Flow:
    1. User enters ingredients into the frontend.
    2. Frontend sends a GraphQL request to the Amplify API.
    3. API triggers a Lambda function.
    4. Lambda function sends a request to Amazon Bedrock with the ingredient list.
    5. Amazon Bedrock processes the request and generates recipes.
    6. Lambda function returns the recipes to the API.
    7. API sends the recipes back to the frontend.
    8. Frontend displays the generated recipes to the user.
